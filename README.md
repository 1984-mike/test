## API Reference

### Calculator Class

- `add(a, b)` - Addition
- `subtract(a, b)` - Subtraction  
- `multiply(a, b)` - Multiplication
- `divide(a, b)` - Division (raises ValueError for division by zero)
- `get_history()` - Returns list of calculation history
- `clear_history()` - Clears calculation history
